# Support

## Getting Help

- **Bug reports**: [Open an issue](https://github.com/EffortlessMetrics/adze/issues/new?template=bug_report.yml)
- **Questions & discussion**: [GitHub Discussions](https://github.com/EffortlessMetrics/adze/discussions)
- **Feature requests**: [Open an issue](https://github.com/EffortlessMetrics/adze/issues/new?template=feature_request.yml)

## Before Opening an Issue

1. Search [existing issues](https://github.com/EffortlessMetrics/adze/issues) to avoid duplicates
2. Check the [documentation](https://github.com/EffortlessMetrics/adze/tree/main/book)
3. Try to create a minimal reproduction

## Security Issues

Please **do not** open public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md) for reporting instructions.
