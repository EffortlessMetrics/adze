## 2026-02-01 - Accessible Tabs in Vanilla JS
**Learning:** Implementing WAI-ARIA tabs in vanilla JS is straightforward but requires careful state management of `aria-selected` and `tabindex` in parallel with visual classes. Roving tabindex is essential for keyboard navigation.
**Action:** For future vanilla JS projects, create a reusable `TabController` class that encapsulates this logic to avoid repetitive DOM manipulation code.
