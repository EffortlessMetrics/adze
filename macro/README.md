# adze-macro

Procedural macros for defining tree-sitter grammars alongside Rust logic.

Provides the `#[adze::grammar]`, `#[adze::language]`, and `#[adze::leaf]`
attribute macros used to annotate Rust types as grammar definitions.

This crate is part of the [Adze](https://github.com/effortlessmetrics/adze)
workspace -- an AST-first grammar toolchain for Rust.
