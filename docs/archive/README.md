# Archived Documentation

These files are preserved for reference but are **no longer maintained**. Content may conflict with current project state.

| `specs/` | Design specifications for GLR engine, arena allocator, parse tables, etc. |
| `plans/` | Investigation findings and implementation plans |
| `implementation/` | Pure-Rust implementation status and roadmap documents |
| `releases/` | Release notes, changelogs, and publish checklists |
| `sessions/` | Session summaries and progress tracking |
| `status/` | Phase completion reports and known-red tracking |
| `roadmaps/` | Historical roadmaps and next-steps documents |
| `findings/` | Investigation reports (e.g., decoder GLR analysis) |

| `implementation-sketches/` | Rust code sketches for scanners, fields, etc. |
| `issues/` | Issue investigation documents |
| `root-level/` | Files that previously lived at the repository root |
