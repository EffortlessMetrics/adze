## 2025-05-22 - Playground Keyboard Shortcuts
**Learning:** Users instinctively reach for `Ctrl+Enter` (or `Cmd+Enter`) in code editors to trigger the primary action (Run/Parse). This pattern is so established in developer tools that its absence interrupts flow.
**Action:** Always implement keyboard shortcuts for the primary "Run" action in any code-based interface and ensure they are discoverable via tooltips or helper text.
