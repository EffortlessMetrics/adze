# adze-bdd-grammar-fixtures

Shared GLR BDD grammar and parse-table fixture helpers.

This crate now composes runtime metadata/token fixtures from
`adze-bdd-runtime-fixtures-core` to keep grammar construction and runtime fixture
responsibilities separate.

Part of the [adze](https://github.com/EffortlessMetrics/adze) workspace.
