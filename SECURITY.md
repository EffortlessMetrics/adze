# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in rust-sitter, please report it
through one of the following channels:

1. **GitHub Security Advisories** (preferred):
   [Report a vulnerability](https://github.com/EffortlessMetrics/rust-sitter/security/advisories/new)

2. **Email**: git@effortlesssteven.com

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or error output

## Response

We will acknowledge receipt within 72 hours and provide an initial assessment
within one week.

## Scope

This project has not yet published stable releases to crates.io. All code on
`main` should be considered pre-release. We still take security seriously and
will address reported issues promptly.
